# Boca++ Privacy Policy

_Last updated: 2026-05-28_

Boca++ is a browser extension that enhances the interface of the BOCA
programming-contest control system.

## What the extension stores

- **The hostname of the server you enable it on** (e.g. `137.184.1.39`). This is
  saved with `chrome.storage.sync` so the extension stays enabled across sessions.
- Some UI preferences (such as run ordering and aliases) are stored locally in your
  browser.

## What the extension does NOT do

- It does **not** collect, transmit, or sell any personal or contest data.
- It does **not** send any data to the extension author or any third party.
- It runs only on the specific server you explicitly enable by clicking the toolbar
  icon, and only on that server's pages.

## Contact

For questions, open an issue on the project's source repository.
